# Signal Vault 텔레그램 스크립트

## 실행 방법

```bash
cd scripts/telegram
pip install -r requirements.txt
python send_sv_message.py
```

가상환경 사용 권장:

```bash
cd scripts/telegram
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python send_sv_message.py
```
