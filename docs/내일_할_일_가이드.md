# 내일 할 일 가이드

어드민 정상화 → 팀 등록 → 실제 데이터 흐름 검증 순서로 진행하세요.

---

## 1. 어드민 정상화 (최우선)

### 1-1. 어드민 이메일 등록
- [lib/admin/admin_config.dart](../lib/admin/admin_config.dart) 열기
- `adminAllowedEmails`에 본인 구글 이메일 추가 (예: `'myname@gmail.com'`)

### 1-2. 웹 구글 로그인 설정
- [docs/웹_구글_로그인_설정.md](웹_구글_로그인_설정.md) 참고
- [web/index.html](../web/index.html)의 `google-signin-client_id` meta 태그에 Web Client ID 입력

### 1-3. 실행 및 확인
```bash
flutter run -d chrome
```
- 브라우저에서 `http://localhost:포트/#/admin` 접속
- 구글 로그인 → 팀 목록/팀 생성/멤버 관리 화면 정상 노출 확인

---

## 2. 팀 등록

- 어드민 사이드바 → **팀 생성** 클릭
- 입력 예시:
  - 팀 ID: `youngwon_fc`
  - 팀명: `영원FC`
  - 지역: `서울 금천구`
  - 팀 소개: (자유 입력)
- **팀 생성** 버튼 클릭
- Firestore에서 `teams/youngwon_fc`, `teams_public/youngwon_fc` 생성 확인 (선택)

---

## 3. 실제 데이터 흐름 검증

1. **앱**: 팀 선택 페이지 → "영원FC" 노출 확인 → **참여하기** → 구글 로그인 → 가입 신청 (status: pending)
2. **어드민**: 멤버 관리 → pending 승인 (status: active)
3. **앱**: 팀 선택 → **들어가기** → 홈 탭 진입
4. **테스트 로그인**: 팀 선택 페이지 **테스트 로그인**으로 샘플 데이터 시딩 후, 일정·매치·투표·회비 등 앱 전체 동작 확인

---

## 4. 검증 체크리스트

- [ ] 팀 선택 → 참여하기 → 승인 → 들어가기 흐름
- [ ] 홈 탭 (매치 카드, 공지, 투표 배너)
- [ ] 일정 탭 (캘린더, 이벤트)
- [ ] 매치 탭 (경기 목록, 참석/불참)
- [ ] MY 탭 (회비, 설정)
