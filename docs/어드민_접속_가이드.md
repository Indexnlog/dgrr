# 어드민 접속 가이드

## 빠른 1페이지 (터미널 없이)

### 누가 어드민 버튼을 보나?
- 팀 선택 화면에서 **운영진 이메일(admin 설정에 등록된 계정)**만 어드민 버튼이 보입니다.
- 일반 멤버 계정에는 어드민 버튼/배너가 노출되지 않습니다.

### 접속 동선
1. 웹 앱 접속
2. 로그인
3. 팀 선택 화면 상단의 **운영진 어드민 접속** 클릭
4. 어드민 로그인 후 `팀 등록` 또는 `가입 승인` 수행

### 가장 자주 쓰는 2가지
- **팀 등록**: 팀 ID / 팀명 / 지역 / 소개 입력 후 생성
- **가입 승인**: pending 멤버를 승인해서 앱 진입 가능 상태로 변경

## 2. 어드민 접속 방법

### 방법 A: 팀 선택 페이지에서 접속
1. 웹 앱 접속 후 로그인
2. 팀 선택 페이지에서 **운영진 어드민 접속** 클릭
3. → `#/admin`으로 이동 → **어드민 로그인 화면** 표시

### 방법 B: 주소창에 직접 입력
1. 브라우저 주소창에 `<웹 앱 주소>/#/admin` 입력 후 엔터
3. → **어드민 로그인 화면** 표시

---

## 3. 어드민 흐름 (처음 접속 시)

1. **로그인 화면** → "구글 로그인" 클릭
2. 구글 계정 선택 (admin_config.dart에 등록된 이메일만 접근 가능)
3. 로그인 성공 → **팀 목록** 화면
4. 사이드바: **팀 목록** | **팀 생성** | **로그아웃**

---

## 4. 팀 등록

1. 사이드바에서 **팀 생성** 클릭
2. 입력:
   - 팀 ID: `youngwon_fc`
   - 팀명: `영원FC`
   - 지역: `서울 금천구`
   - 팀 소개: (자유 입력)
3. **팀 생성** 버튼 클릭
4. → `teams/youngwon_fc` + `teams_public/youngwon_fc` 생성됨
5. 앱 팀 선택 페이지에서 "영원FC" 노출 확인

---

## 5. 가입 승인

1. 앱에서 "영원FC" → **참여하기** → 구글 로그인 → 가입 신청
2. 어드민 → **팀 목록** → "영원FC" 클릭 → **멤버 관리**
3. pending 상태 멤버 → **승인** 클릭
4. 앱에서 "들어가기" 가능
