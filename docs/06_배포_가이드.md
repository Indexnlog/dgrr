# 배포 가이드

> 안드로이드 APK 빌드 및 스토어 배포

---

## 1. 안드로이드 APK (테스트용)

### 디버그 APK (빠른 테스트)
```bash
flutter build apk --debug
```
- 출력: `build/app/outputs/flutter-apk/app-debug.apk`
- 실기기에 USB 연결 후 `adb install` 또는 파일 전송 후 설치

### 릴리즈 APK
```bash
flutter build apk --release
```
- 서명 설정 필요 (`android/app/build.gradle.kts`)

---

## 2. 안드로이드 AAB (Play Store)

```bash
flutter build appbundle --release
```
- 출력: `build/app/outputs/bundle/release/app-release.aab`
- Play Console에서 업로드

---

## 3. iOS (App Store)

- Apple Developer 계정 ($99/년) 필요
- Xcode에서 서명/프로비저닝 설정
- `flutter build ios --release` 후 Xcode로 아카이브 → 업로드

---

## 4. 환경 변수 / 설정

- `google-services.json` (Android), `GoogleService-Info.plist` (iOS) 필수
- Firebase 프로젝트에 앱 등록 후 다운로드

---
