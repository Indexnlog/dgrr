## 2. 기술 스택

| 영역 | 기술 |
|------|------|

---

## 3. 폴더 구조

```
lib/
├── app/                    # 앱 전역 (라우터, 위젯)
├── core/                   # 공통 (테마, 권한, 에러)
└── features/
    ├── auth/               # 인증
    ├── events/             # 수업/이벤트
    ├── fees/               # 회비
    ├── matches/            # 경기
    ├── mypage/             # MY 페이지
    ├── onboarding/         # 팀 선택
    ├── polls/              # 투표
    ├── posts/              # 공지
    ├── reservations/       # 구장 예약
    ├── schedule/           # 일정
    └── teams/              # 팀/멤버
```

---

## 4. 주요 비즈니스 로직

- **회비제**: 월별 등록 투표(20~24일), 일자별 참석 투표(25일~)
- **경기 확정**: `attendees >= 7` AND `opponent.status == confirmed` 시 카드 노출
- **실시간 기록**: 골/교체 1탭 빠른 기록 → Firestore 즉시 반영
- **온보딩**: `teams_public` 검색 → 가입 요청 → Google 로그인
