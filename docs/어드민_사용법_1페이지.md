# 어드민 사용법 1페이지

## 목적
- 운영진(관리자)만 빠르게 접속해서 `팀 등록`, `가입 승인`을 처리합니다.
- 일반 사용자 동선과 분리해 운영 화면 노출을 최소화합니다.

## 접속 조건
- 팀 선택 화면에서 어드민 버튼은 **허용된 운영진 이메일**에만 노출됩니다.
- 일반 멤버 계정에서는 버튼이 보이지 않습니다.

## 접속 순서 (터미널 불필요)
1. 웹 앱 접속 후 로그인
2. 팀 선택 화면 상단 `운영진 어드민 접속` 클릭
3. 어드민 로그인
4. 필요한 작업 실행

## 핵심 작업
- 팀 등록
  - 팀 ID, 팀명, 지역, 소개 입력 후 생성
- 가입 승인
  - pending 멤버를 승인 처리

## 운영 체크
- 승인 후 앱에서 멤버가 `들어가기` 가능한지 확인
- 잘못 승인한 경우 상태를 즉시 점검
- 긴급 이슈 시 신규 승인 일시 중단 후 공지
