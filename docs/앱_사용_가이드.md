# 영원FC 앱 사용 가이드

> 기능이 많아서 정리가 안 될 때 보는 문서

---

## 1. 한눈에 보기

```
[신규] 앱 설치 → 팀 검색 → 참여하기 → 구글 로그인 → 승인 대기
         ↓
[승인됨] 홈 화면 (4탭)
         ├── 홈: 오늘 경기, 참석/불참 투표
         ├── 일정: 수업·모임, 예약 공지, 투표
         ├── 매치: 경기 목록, 상대팀, 골/교체 기록
         └── MY: 프로필, 회비, 영원FC 안내

[관리자] 웹 어드민 (localhost/admin)
         ├── 팀 생성
         ├── 가입 신청 승인/거절
         └── 멤버 관리
```

---

## 2. 역할별로 뭘 하나요?

### 일반 멤버
| 할 일 | 어디서 |
|-------|--------|
| 월별 등록 투표 (등록/휴회/미등록) | MY → 등록 카드 |
| 일자별 수업 참석 투표 | 일정 → 수업 상세 |
| 경기 참석/불참 투표 | 홈 or 매치 → 경기 상세 |
| 공 가져가기 | 경기 상세 |
| 회비 입금 | 모임통장 (총무 공유) |

### 총무
| 할 일 | 어디서 |
|-------|--------|
| 회비 관리 | MY → 회비 관리 |
| 미납자 확인 | 회비 관리 페이지 |
| 입금 확인 후 isPaid 체크 | 회비 관리 |

### 운영진/코치
| 할 일 | 어디서 |
|-------|--------|
| 경기 생성 | 매치 → + 버튼 |
| 상대팀 관리 | 매치 → 상대팀 |
| 라인업 편성 | 경기 상세 → 라인업 |
| 골/교체 기록 | 경기 상세 (경기 중) |
| 수업 등록 | 일정 (수업 생성) |
| 예약 공지 작성 | 일정 → 예약 공지 |
| 투표 생성 | 일정 → 투표 |
| 구장 관리 | MY → 구장 관리 |

## 3. 탭별 상세

### 홈 탭
- **오늘/라이브 경기 카드**: 참석 7명 이상 + 상대 확정 시만 표시
- **참석/불참 버튼**: 탭하면 바로 반영 (Optimistic UI)
- **공 가져가기**: "저도 들고가요" 체크
- **게시글**: 공지·소식 (홈 → 게시글)

### 일정 탭
- **캘린더**: 수업·모임·경기 일정
- **수업 상세**: 참석/지각/불참 투표
- **예약 공지**: 월·목 구장 예약 안내 (운영진)
- **투표**: 월별 등록, 일정 투표 등

### 매치 탭
- **경기 목록**: 성사된 경기만 (참석 7명+ 상대 확정)
- **경기 상세**: 참석/불참, 라인업, 골/교체 기록
- **상대팀 관리**: 상대팀 목록 (운영진)

### MY 탭
- **프로필**: 사진, 이름
- **월별 등록**: 20~24일 투표, 25일~말일 일자별 참석
- **회비 관리**: 총무 전용
- **영원FC 안내**: 회칙·회비·구장 (온보딩 가이드)
- **구장 관리**: 운영진 전용

---

## 4. 월별 사이클 (언제 뭘 하나요?)

| 시기 | 누가 | 뭘 해요 |
|------|------|---------|
| **20~24일** | 전체 | 앱에서 익월 등록 투표 (등록 5만/휴회 2만/미등록 0) |
| **25일~말일** | 전체 | 일자별 수업 참석 가능 일자 체크 |
| **매달** | 등록·휴회 회원 | 모임통장 입금 |
| **매달** | 총무 | 입금 확인 후 isPaid 체크 |
| **매주 월** | 운영진 | 구장 예약 참여 (예약 공지) |

## 5. 어드민 웹 (개발자/관리자)

```bash
flutter run -d chrome
# 브라우저에서 http://localhost:xxxxx/admin
```

| 기능 | 경로 |
|------|------|
| 로그인 | /admin/login |
| 팀 목록 | /admin/teams |
| 팀 생성 | /admin/teams/create |
| 멤버 관리 (승인/거절) | /admin/teams/:teamId/members |

## 6. 처음 켜는 순서 (체크리스트)

### 1) 어드민에서 팀 만들기
1. 웹에서 `/admin` 접속
2. 구글 로그인
3. 팀 생성 → 팀 ID, 팀명, 지역, 소개 입력

### 2) 신규 멤버 온보딩
1. 앱 설치
2. 팀 선택 화면 → "영원FC 안내" 먼저 읽기 (선택)
3. 팀 검색 → 참여하기 → 구글 로그인
4. 어드민에서 가입 신청 승인

### 3) 승인된 멤버
1. 앱 다시 열면 홈으로 자동 이동
2. MY → 월별 등록 투표 (해당 시기면)
3. 일정에서 수업 참석 투표
4. 홈/매치에서 경기 참석 투표

---

## 7. 자주 묻는 것

**Q. 경기 카드가 안 보여요**  
→ 참석 7명 이상 + 상대팀 확정이어야 표시됩니다.

**Q. 월별 등록 투표가 안 보여요**  
→ 20~24일에만 MY 탭에 표시됩니다.

**Q. 수업 참석 투표는?**  
→ 25일~말일, 일정 탭에서 수업 들어가서 투표.

**Q. 운영시트는?**  
→ 앱으로 대체됨. 사용 안 함.

**Q. 회비 입금은?**  
→ 모임통장 (총무가 링크 공유). 입금 후 총무가 앱에서 isPaid 체크.
