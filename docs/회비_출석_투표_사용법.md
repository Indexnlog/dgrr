# 회비·출석 투표 사용법

## 개요

- **월별 등록 투표**: 매월 20~24일 (다음 달 등록/휴회/미등록 선택)
- **일자별 참석 투표**: 매월 25일~말일 (수업 참석 가능 일자 체크)

---

## 1. 진입 경로

| 위치 | 경로 |
|------|------|
| 홈 | 활성 투표 있으면 "회비·출석 투표 N건 진행중" 배너 → 탭 |
| 일정 | 상단 "회비·출석 투표" 버튼 |
| 상단 바 | 벨 아이콘 → 투표 목록 |

---

## 2. 운영진: 투표 생성

1. **일정** 탭 → **회비·출석 투표** → **+** (우측 상단)
2. **투표 유형** 선택:
   - **월별 등록**: 다음 달 등록(5만)/휴회(2만)/미등록(0원)
   - **일자별 참석**: 해당 월 수업 일정 중 참석 가능 날짜 복수 선택
3. **대상 월** 선택 후 **투표 만들기**

---

## 3. 멤버: 투표하기

1. 홈/일정에서 투표 배너 또는 버튼 탭
2. 투표 카드 선택 → 상세 화면
3. **월별 등록**: 등록/휴회/미등록 중 1개 선택 (변경 가능)
4. **일자별 참석**: 참석 가능 날짜 복수 체크

> 투표 시 **Registration**에 자동 반영됨 (회비 산정 기준)

---

## 4. 출석 투표 → 수업 일정 생성

1. 일자별 참석 투표 **종료 후**
2. 운영진/코치가 투표 상세 화면에서 **수업 일정 생성** 버튼 탭
3. 투표 결과 기반으로 `events` 컬렉션에 수업 일정 생성
